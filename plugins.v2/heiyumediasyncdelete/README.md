# 媒体文件同步删除

基于 `RemoveLink / 清理媒体文件` 2.12 的个人定制版，使用独立插件 ID：`HeiyuMediaSyncDelete`。

## 功能

- 监控媒体库文件删除事件。
- 按硬链接 inode 同步删除源文件或目标文件。
- 支持延迟确认，避免 MoviePilot 重新整理时误删。
- 可选清理刮削文件、空目录、转移记录。
- 可选联动下载任务/种子删除。
- 在插件详情页记录类似 `媒体文件同步删除` 的历史卡片，展示触发路径、删除的硬链接、联动操作、跳过原因和错误。

## 建议配置

先验证文件清理，再开启删种：

```text
发送通知：开启
联动删除种子：先关闭
删除历史记录：先关闭
延迟删除：开启
延迟秒数：30
```

监控目录示例：

```text
/volume1/video/电视剧
/volume1/video/电影
/volume1/video/link
```

确认 Jellyfin 删除媒体后，详情页历史和日志都符合预期，再开启联动删除种子。
